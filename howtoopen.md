你可以用這套最穩定流程操作（你目前環境建議用 npx pnpm@latest）：

啟動 Dev 模式
1. 進入資料夾  
D:\a_114-2_classHW\thu_web_HW\portfolio-lee359
2. 執行  
npx pnpm@latest dev
3. 打開網址  
http://localhost:3000/

關閉 Dev 模式
1. 在同一個正在跑 dev 的終端機按 Ctrl + C
2. 若詢問是否中止，輸入 Y 再按 Enter

啟動 Preview 模式
1. 進入資料夾  
D:\a_114-2_classHW\thu_web_HW\portfolio-lee359
2. 先建置  
npx pnpm@latest build
3. 再啟動預覽  
npx pnpm@latest preview
4. 打開網址  
http://localhost:4173/

關閉 Preview 模式
1. 在同一個正在跑 preview 的終端機按 Ctrl + C
2. 若詢問是否中止，輸入 Y 再按 Enter

補充
1. 如果你之後已安裝好全域 pnpm，上面指令可把 npx pnpm@latest 換成 pnpm。
